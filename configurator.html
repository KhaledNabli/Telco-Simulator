Demo Configuration


This demo has been developed by Khaled Nabli and Mathias Bouten .


This Demo Configurator can be used to change the content of this demo.

Profile Customer profile content for the Advisor Frontend of this demo
Analytics Customer analytics content for the analytics bars in the Advisor Frontend
SMS SMS recipients can be managed here to send out SMS when offers are accepted in the Advisor Frontend
Action History Recent customer actions shown in the Action History part of the Advisor Frontend
Next Best Actions New offer recommendations for the customer displayed in the Advisor Frontend
Search Post History Search strings and post contents of the customer displayed in the Advisor Frontend
Images You can configure which images should appear in the poster and the iphone area of the demo
