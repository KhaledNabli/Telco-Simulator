Data fields

You can define the data fields which will be usable in the demo. The data fields can be passed to an ESP variable in the Window defined for location events. Furthermore, the data fields can be retrieved and updated from an RTDM variable.

Entity Variable Label ESP Variable RTDM Variable
Customer id Customer ID customerId default attribute
Customer label Customer Name customerNm default attribute
Location id Location ID locationId default attribute
Location label Location Name locationNm default attribute
Event eventDttm Event Timestamp eventDttm default attribute
